# Mi Proyecto

Este README contiene las instrucciones para instalar las dependencias necesarias para ejecutar el proyecto.

## Dependencias

A continuación, se enumeran las dependencias utilizadas en el proyecto junto con sus versiones:

- alembic: 1.12.0
- Flask: 2.3.3
- Flask-Login: 0.6.2
- Flask-Migrate: 4.0.5
- Flask-SQLAlchemy: 3.0.5
- pytz: 2023.3.post1
- reportlab: 4.0.6
- SQLAlchemy: 2.0.20
- Werkzeug: 2.3.7
... [y así sucesivamente para todas las dependencias que utilizas]

## Instalación

Para instalar todas las dependencias necesarias para el proyecto, puedes usar el siguiente comando:

```bash
pip install alembic==1.12.0 Flask==2.3.3 Flask-Login==0.6.2 Flask-Migrate==4.0.5 Flask-SQLAlchemy==3.0.5 pytz==2023.3.post1 reportlab==4.0.6 SQLAlchemy==2.0.20 Werkzeug==2.3.7
